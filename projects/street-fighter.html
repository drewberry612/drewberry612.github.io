Street Fighter Clone

View on GitHub

This is a clone of Street Fighter made from scratch in Java. The game features a variety of attacks, each with unique sound effects, damage, and stunning effects. The goal is to knock out the other player, and there are multiple playable characters to choose from. The game includes retro visuals, a simple AI opponent, and multiple playable characters.

Gameplay

The game includes two primary modes:

  • Player vs Player (PvP): Both players control characters using the same keyboard, configured in the options menu.
  • Player vs AI (Story Mode): In this mode, you battle against a simple AI that uses random number generation to decide which moves to use. The AI is erratic, making it hard to predict and defeat. It only uses moves that are consistent with its current stance (e.g., no jumping attack while crouched).

Best of 3: The first player to knock out the other twice wins the match.

Attacks

There are 3 types of attacks in the game:

  • Light
  • Medium
  • Heavy

These attacks can be used while:

  • Standing
  • Crouched
  • Moving Forward
  • Moving Backward
  • Jumping

Each action results in different effects, ranges, and speeds, making the gameplay dynamic and fun.
