Angry Birds Clone

View on GitHub

For this project, I created an Angry Birds clone using Java and JBox2D. The game follows the same core mechanics as the popular mobile game, where players use a slingshot and birds to destroy structures housing green pigs that have stolen their eggs.

Gameplay Video

Key Features

  • 3 Levels: The game includes 3 levels designed by me, showcasing the core gameplay mechanics of Angry Birds. Each level increases in complexity, providing a fun and challenging experience.
  • Custom Physics Engine: I implemented my own bespoke physics engine using JBox2D. I utilised a ContactListener to create a realistic interaction between the slingshot, birds, and destructible objects. This makes the gameplay feel almost identical to the original game.
  • Bird Types: There are 4 types of birds, each with their own unique abilities. These birds can be launched using the slingshot to hit targets, and each one impacts the structures differently.
  • Pig Types: I included 2 types of pigs, with varying resistance to destruction. The pigs act as the primary targets for the player.
  • Destructible Materials: The game has 3 types of materials for building structures, each with its own unique properties, such as wood, stone, and ice.
  • Explosives & Ropes: Explosives can be triggered by hitting them at the right speed, while ropes allow for more complex and creative structures to be built by the pigs.
  • Scoring System: The game encourages efficiency, with a scoring system that rewards players for using fewer birds to destroy all the pigs. A timer tracks the optimal time for each level, promoting competitive play.

Audio and Visuals

  • Sounds: The game includes sound effects taken from the original Angry Birds game, playing when collisions occur or actions are made by the player.
  • Visuals: I recreated retro-style visuals, giving the game a similar look and feel to the original while adding my own touch for a unique experience.
